# RockPaperScissors

Un clásico juego de Piedra, Papel o Tijeras desarrollado con HTML, CSS y JavaScript.

## Funcionalidades

- **Juego contra la CPU**: El jugador elige una opción y la CPU selecciona una aleatoriamente.
- **Marcador en tiempo real**: Se muestra el puntaje actual del jugador y de la CPU.
- **Indicador de ganadores por ronda**:
  - Se muestra quién ganó cada ronda con mensajes de colores:
    - Verde: Gana el Jugador.
    - Rojo: Gana la CPU.
    - Amarillo: Empate.
- **Fin del juego**: El primero en llegar a 3 puntos gana la partida total.
- **Reinicio**: Botón para reiniciar el juego y los marcadores en cualquier momento.

## Cómo jugar

1. Selecciona una de las opciones: Piedra, Papel o Tijeras.
2. Espera a que termine la animación para ver qué eligió la CPU.
3. El marcador se actualizará automáticamente.
4. ¡Gana 3 rondas para vencer a la CPU!
